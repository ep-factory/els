Saves the uploaded file.

This method can throw exceptions if there is a problem when saving the file.

If you don't pass a file name, it will be generated by the generateFilename method. This will only work if you have passed a path when initializing this instance.

Paramètres:
string $file The file path to save the file
int $fileMode The octal mode to use for the new file
