__construct ( $  data = array()  ) 

Initializes a new writable instance of the sfDateTimeFormatInfo class that is dependent on the ICU data for date time formatting information. N.B.You should not initialize this class directly unless you know what you are doing. Please use use sfDateTimeFormatInfo::getInstance() to create an instance.

Paramètres:
array $data ICU data for date time formatting.
