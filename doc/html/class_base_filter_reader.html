Documentation des constructeurs et destructeur

__construct ( $  in = null  ) 

Constructor used by Phing's introspection mechanism. The original filter reader is only used for chaining purposes, never for filtering purposes (and indeed it would be useless for filtering purposes, as it has no real data to filter). ChainedReaderHelper uses this placeholder instance to create a chain of real filters.
