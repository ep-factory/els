Deploys a set of files to a remote FTP server.

Example usage: <ftpdeploy host="host" port="21" username="user" password="password" dir="public_html" mode="ascii" clearfirst="true"> <fileset dir="."> <exclude name="phing"> <exclude name="build.xml"> <exclude name="images/**.png"> <exclude name="images/**.gif"> <exclude name="images/**.jpg"> </fileset> </ftpdeploy>
