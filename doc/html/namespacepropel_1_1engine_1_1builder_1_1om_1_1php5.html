Detailed Description

Generates the empty PHP5 stub node object class for user object model (OM).

This class produces the empty stub class that can be customized with application business logic, custom behavior, etc.

This class replaces the ExtensionNode.tpl, with the intent of being easier for users to customize (through extending & overriding).
