static reset (  )  [static]

Resets any static variables between building a SQL file for a database.

Theoretically, Propel could build multiple .sql files for multiple databases; in many cases we don't want static values to persist between these. This method provides a way to clear out static values between iterations, if the subclasses choose to implement it.
