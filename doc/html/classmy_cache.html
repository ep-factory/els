Initializes this sfCache instance.

Parameters:

Available options:

* automatic_cleaning_factor: The automatic cleaning process destroy too old (for the given life time) (default value: 1000) cache files when a new cache file is written. 0 => no automatic cache cleaning 1 => systematic cache cleaning x (integer) > 1 => automatic cleaning randomly 1 times on x cache write

* lifetime (optional): The default life time (default value: 86400)
