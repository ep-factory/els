Initializes a new writable instance of the sfNumberFormatInfo class that is dependent on the ICU data for number, decimal, and currency formatting information. N.B.You should not initialize this class directly unless you know what you are doing. Please use use sfNumberFormatInfo::getInstance() to create an instance.
