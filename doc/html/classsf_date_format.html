getPattern ( $  pattern  ) 

Gets the pattern from DateTimeFormatInfo or some predefined patterns. If the $pattern parameter is an array of 2 element, it will assume that the first element is the date, and second the time and try to find an appropriate pattern and apply DateTimeFormatInfo::formatDateTime See the tutorial documentation for futher details on the patterns.
