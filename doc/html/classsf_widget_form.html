generateId ( $  name,
$  value = null 
)

Returns a formatted id based on the field name and optionally on the field value.

This method determines the proper form field id name based on the parameters. If a form field has an array value as a name we need to convert them to proper and unique ids like so:

<samp> name[] => name (if value == null) name[] => name_value (if value != null) name[bob] => name_bob name[item][total] => name_item_total </samp>

This method also changes all invalid characters to an underscore (_):

Ids must begin with a letter ([A-Za-z]) and may be followed by any number of letters, digits ([0-9]), hyphens ("-"), underscores ("_"), colons (":"), and periods (".").

Parameters:
string$nameThe field name
string$valueThe field value
Returns:
string The field id or null.
