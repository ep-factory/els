addClassBody ( &$  script  )  [abstract, protected]

This method adds the contents of the generated class to the script.

This method is abstract and should be overridden by the subclasses.

Hint: Override this method in your subclass if you want to reorganize or drastically change the contents of the generated peer class.

Paramètres:
