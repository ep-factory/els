PHP error handler send PHP errors to log.

PHP user space error handler can not handle E_ERROR, E_PARSE, E_CORE_ERROR, E_CORE_WARNING, E_COMPILE_ERROR, E_COMPILE_WARNING, and most of E_STRICT.

Parameters:
string$errnoThe level of the error raised, as an integer.
string$errstrThe error message, as a string.
string$errfileThe filename that the error was raised in, as a string.
string$errlineThe line number the error was raised at, as an integer.
array$errcontextAn array that points to the active symbol table at the point the error occurred.
