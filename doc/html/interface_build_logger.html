setMessageOutputLevel ( $  level)

Sets the min log level that this logger should respect.

Messages below this level are ignored.

Constants for the message levels are in Project.php. The order of the levels, from least to most verbose, is:

  • Project::MSG_ERR
  • Project::MSG_WARN
  • Project::MSG_INFO
  • Project::MSG_VERBOSE
  • Project::MSG_DEBUG
Parameters:
int$levelThe log level integer (e.g. Project::MSG_VERBOSE, etc.).

Implemented in XmlLogger, and DefaultLogger.
