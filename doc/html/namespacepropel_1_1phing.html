Dumps the contenst of selected databases to XML data dump file.

The generated XML files can have corresponding DTD files generated using the PropelDataDTDTask. The results of the data dump can be converted to SQL using the PropelDataSQLTask class.

The database may be specified (via 'databaseName' attribute) if you only want to dump the contents of one database. Otherwise it is assumed that all databases described by datamodel schema file(s) will be dumped.
