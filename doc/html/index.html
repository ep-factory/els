CKEditor - PHP server side intergation

CKEditor

Visit CKEditor web site to find more information about the editor.

Installation

Include ckeditor.php in your PHP web site.

 <?php
 include("ckeditor/ckeditor.php");
 ?>

Create CKEditor class instance and use one of available methods to insert CKEditor.

 <?php
 $CKEditor = new CKEditor();
 echo $CKEditor->textarea("field1", "<p>Initial value.</p>");
 ?>
